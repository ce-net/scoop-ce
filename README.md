# scoop-ce

Scoop bucket for [CE](https://github.com/ce-net/ce) — peer-to-peer compute mesh and economy.

## Install

```powershell
scoop bucket add ce-net https://github.com/ce-net/scoop-ce
scoop install ce
```

## Updating

```powershell
scoop update ce
```

Hashes are auto-updated from `sha256sums.txt` in each GitHub release via the `autoupdate` config.
